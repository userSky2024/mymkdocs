    第一部分：Java简介和开发环境搭建

    1.1 Java简介

    ‌Java是什么‌：Java是一种面向对象的、基于类的、通用型、跨平台的高级程序设计语言。 ‌Java的特点‌：平台无关性（一次编写，到处运行）、安全性、自动垃圾回收等。

    1.2 安装Java开发环境

    ‌下载JDK‌：到Oracle官网下载适合你操作系统的Java Development Kit (JDK)。 ‌安装JDK‌：按照提示安装即可，并配置环境变量（JAVA_HOME和PATH）。

    1.3 安装IDE（集成开发环境）

    ‌推荐IDE‌：IntelliJ IDEA、Eclipse、NetBeans等。 ‌安装和配置‌：下载并安装你选择的IDE，配置JDK路径。
