    hyprland桌面环境安装

    在Arch Linux上安装Hyprland的步骤如下：

    0.安装kitty终端

    1
    sudo pacman -Syu kitty
    1. 安装基础工具：确保你的系统是基于Arch或其派生版，并已安装 base-devel 套件，用于编译软件。 
    1
    sudo pacman -Syu base-devel
    2. 获取Arch-Hyprland自动化安装工具：克隆Arch-Hyprland仓库到本地，并运行安装脚本。 
    1
2
3
    git clone https://github.com/JaKooLit/Arch-Hyprland.git
cd Arch-Hyprland
sh install.sh
    这个自动化工具会帮助你快速安装Hyprland，并提供配置Rofi、Waybar等组件的选项。

    3. 如果不使用自动化脚本，安装Hyprland： sudo pacman -S hyprland 4. 配置Hyprland：安装完成后，你可能需要配置Hyprland。Hyprland的配置文件通常位于 ~/.config/hypr/hyprland.conf 。如果这个文件不存在，你需要创建它，并可能需要从样例配置文件中复制内容到这个文件中。 /example/hyprland.conf 

    1
    git clone --recursive https://github.com/hyprwm/Hyprland

    5. 启动Hyprland：安装并配置完成后，重新登录系统，Hyprland应该会自动启动。如果没有自动启动，可以手动运行： hyprland 来启动Hyprland。 以上步骤应该可以帮助你在Arch Linux上安装并启动Hyprland。如果在安装过程中遇到任何问题，可以参考Hyprland的官方文档或社区论坛获取更多帮助。

    以下是配置文件中定义的快捷键及其功能：

    快捷键组合 功能描述
    Super + Q 打开终端（kitty）
    Super + C 关闭活动窗口
    Super + M 退出 Hyprland
    Super + E 打开文件管理器（dolphin）
    Super + V 切换浮动窗口
    Super + R 打开菜单（wofi）
    Super + P 切换伪平铺模式（dwindle）
    Super + J 切换分割模式（dwindle）
    Super + 左箭头 将焦点移动到左侧窗口
    Super + 右箭头 将焦点移动到右侧窗口
    Super + 上箭头 将焦点移动到上方窗口
    Super + 下箭头 将焦点移动到下方窗口
    Super + 数字键（1-10） 切换到对应的工作区
    Super + Shift + 数字键（1-10） 将活动窗口移动到对应的工作区
    Super + S 切换到特殊工作区（scratchpad）
    Super + Shift + S 将活动窗口移动到特殊工作区（scratchpad）
    Super + 鼠标滚轮向下 切换到下一个工作区
    Super + 鼠标滚轮向上 切换到上一个工作区
    Super + 鼠标左键拖动 移动窗口
    Super + 鼠标右键拖动 调整窗口大小
    音量增大键 增加音量
    音量减小键 减小音量
    音量静音键 切换音量静音
    麦克风静音键 切换麦克风静音
    亮度增加键 增加亮度
    亮度减小键 减小亮度
    媒体播放键 播放/暂停媒体
    媒体下一曲键 播放下一曲
    媒体上一曲键 播放上一曲
