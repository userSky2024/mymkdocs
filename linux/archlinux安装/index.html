    archlinux安装

    1小时40分钟 最近使用1个小时安装完成

    下载镜像 安装指南 - Arch Linux 中文维基

    使用 图吧工具箱 其他工具-rufus 制作启动盘

    重启 进入bios 关闭安全启动 选择u盘启动

    进入之后...

    字体大小 setfont ter-132b

    连接wifi # iwctl 列出所有WiFi设备 # device list 扫描网络 # station[网卡名字] scan 列出网络 # station [网卡名字] get-networks 连接wifi # station[网卡名字]connect[wifi名称] 退出 # exit 测试网络 # ping www.baidu.com 更新系统时间 # timedatectl 查看自己分区 # fdisk -l或lsblk 分区 # cfdisk [你的硬盘]

    进入cfdisk
Delete：删除分区
New：创建新分区
Quit:退出
Type：类型
Write：写入
Dump：排序

分4个分区
EFI分区 (100M) FEI Systen
根分区 Linux filesysten(默认)
home分区 Linux hone
交换分区 Linux suap

    格式化分区 (根分区、home分区) # mkfs.ext4 /dev/[分区编号] (交换分区) # mkswap /dev/[交换空间分区] (EFI分区) # mkfs.fat -F 32 /dev/[EFI分区]

    挂载分区 (根分区) # mount /dev/[根分区编号] /mnt (home分区) # mount-mkdir /dev/[home分区编号］/mnt/home (EFI分区) # mount--mkdir /dev/[efi分区编号] /mnt/efi (交换分区) # swapon /dev/[交换空间分区编号]

    配置国内镜像源 # nano /etc/pacman.d/mirrorlist 添加(清华源)

    Server = https://mirrors.tuna.tsinghua.edu.cn/archlinux/$repo/os/$arch

    刷新 # pacman -Syy 导入密钥 # sudo pacman -S archlinux-keyring

    安装基本系统 # pacstrap /mnt [软件包]

    软件包列表


base base-devel 基本系统
linux 
linux-headers [标准内核]
 或linux-ltslinux-lts-headers [长期支持内核]
 或linux-zen-headers [高性能内核]等 linux内核和头文件
linux-firmware (linux固件）
networkmanager (网络)
grub 引导
os-prober (多系统检测)
efibootmgr (efi启动项管理)
intel-ucode (英特尔) 或 amd-ucode (amd) cpu编码 根据自己的cpu选择
bluez bluez-utils (蓝牙)
nano (文本编辑器)

    创建fstab（自动挂载配置文件） # genfstab -U /mnt >> /mnt/etc/fstab

    chroot进入系统 # arch-chroot /mnt 设置时区 # ln -sf /usr/share/zoneinfo/Asia/shanghai /etc/localtime 设置硬件时间 # hwclock --systohc

    本地化 字符集 # nano /etc/locale.gen

    删除en_US.UTF-8 UTF-8和zh_CN.UTF-8 UTF-8前的“#”

    生成locale # locale-gen 语言环境 # nano /etc/locale.conf

    添加 LANG=en_US.UTF-8

    设置主机名 # nano /etc/hostname (主机名输入到文件里) 设置root密码 # passwd 创建普通用户 # useradd -m -G wheel [用户名] 为用户创建密码 # passwd [用户名] 编辑sudoers赋予用户root权限 # nano /etc/sudoers

    [删除%wheel ALL=(ALL:ALL) ALL前的“#”]

    启用服务 # systemctl enable NetworkManager(网络) # systemctl enable bluetooth (蓝牙) 编辑grub # nano /etc/default/grub

    （去掉GRUB_DISABLE_OS_PROBER=false前的"#")

    安装grub服务 # grub-install --target=x86_64-efi --efi-directory=/efi --bootloader-id=grub 如果报efibootmgr未找到 使用命令 pacman -S efibootmgr

    更新grub引导 # grub-mkconfig -o /boot/grub/grub.cfg 安装桌面环境kde # pacman -S xorg plasma 开机自启显示管理 # systemctl enable sddm

    安装工具 # pacman -S [软件包]

    konsole (终端)
dolphin (文件管理器)
ark (解压缩软件)
kate (文本编辑器)

    退出系统 # exit 重启 # reboot

    进入桌面后ctrl+alt+T打开终端 更新grub # sudo grub-mkconfig -o /boot/grub/grub.cfg

    注意这里先连上网 安装中文字体 # sudo pacman -S adobe-source-han-sans-cn-fonts 在设置里设置一下语言,然后重启

    配置国内源 sudo nano /etc/pacman.conf

    开启32为库
去掉[multilib]及下一行前的 '#'

然后下面添加
[archlinuxcn]
Server = https://mirrors.tuna.tsinghua.edu.cn/archlinuxcn/$arch (清华源)
 或 Server = https://mirrors.ustc.edu.cn/archlinuxcn/$arch (中科大源)
 或 Server = https://repo.huaweicloud.com/archlinux/$repo/os/$arch (华为源)

    刷新软件包 # sudo pacman -Syy 导入密钥 # sudo pacman -S archlinuxcn-keyring

    安装后端程序(可选) # sudo pacman -S archlinux-appstream-data packagekit-qt5 fwupd

    安装aur助手yay # sudo pacman -S yay

    安装中文输入法

    # sudo pacman -S fcitx5-im
# sudo pacman -S fcitx5-chinese-addons fcitx5-rime
# yay -S fcitx5-input-support

    重启
